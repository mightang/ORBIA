# HEXFIELD: Galactic Restoration

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | HEXFIELD: Galactic Restoration |
| 장르 | 퍼즐 / 논리 / 전략 |
| 개발 환경 | Python 3.11, pygame |
| 타겟 플랫폼 | PC (Windows / macOS) |
| 주요 특징 | 육각형 타일 기반 지뢰찾기, 은하 탐사형 스테이지 구조, 점진적 난이도 상승, 파스텔톤 SF UI |


## 2. 기획 의도

지뢰찾기는 완벽한 논리 퍼즐의 형태로 오랜 세월 사랑받은 게임이다.  
본 프로젝트는 이를 육각형 격자와 SF 세계관으로 재해석하여,  
논리적 사고의 쾌감과 시각적 몰입을 동시에 제공하는 은하 복원 퍼즐 게임을 구현한다.

**핵심 콘셉트**  
플레이어는 ‘은하 복원자(Galactic Restorer)’가 되어,  
오염된 데이터 필드(지뢰 구역)를 하나씩 정화하며 은하의 질서를 되찾는다.


## 3. 주요 시스템 개요

### 3.1 기본 규칙
- 육각형 타일 기반 지뢰찾기 구조
- 각 타일은 여섯 방향의 인접 타일을 가짐
- 숫자는 주변 블랙홀(지뢰)의 개수를 의미
- 잘못 클릭 시 게임 오버 (체력 시스템 등은 추후 추가 가능)

### 3.2 스테이지 구조
- 전체 맵은 중심부 착륙 지점을 기준으로 육각형 링 구조로 구성
- 링별 난이도 상승
- 바깥으로 갈수록 오염이 심해지고 퍼즐이 복잡해짐

| 구역 | 링 번호 | 난이도 | 스테이지 수 | 설명 |
|------|----------|----------|--------------|------|
| 착륙 지점 | 0 | 튜토리얼 | 1 | 기본 조작, 규칙 설명 |
| Inner Ring | 1 | 쉬움 | 6 | 단순 논리 규칙 중심 |
| Mid Ring | 2 | 보통 | 12 | 복합 패턴 등장 |
| Outer Ring | 3 | 어려움 | 18 | 고난이도, 방해 요소 추가 |
| **총합** | — | — | **37 스테이지** | — |


## 4. 난이도 설계 및 진행 구조

- 튜토리얼: 조작과 기본 규칙 학습  
- 쉬움 구역: 확정적 추론 중심  
- 보통 구역: 불투명 영역, 복합 규칙 등장  
- 어려움 구역: 대형 퍼즐, 고밀도, 제한된 정보  
- 클리어 시 중심에서 정화 파동이 퍼지며 주변 링이 해금됨


## 5. 시각 및 사운드 콘셉트

| 항목 | 콘셉트 |
|------|---------|
| 컬러 팔레트 | 파스텔톤 SF (하늘빛, 보라, 시안 중심) |
| 비주얼 톤 | 차갑지 않은 우주, 데이터 정화와 빛의 확산 이미지 |
| UI 스타일 | 둥근 모서리 육각 타일, 라디얼 글로우, 부드러운 트윈 이징 |
| 애니메이션 | 타일 열림(220ms), 파동 전파(25ms 지연), 정화 이펙트 |
| 사운드 디자인 | 저주파 신스, 하프톤 클릭음, 데이터 복원 효과음 |


## 6. 개발 단계 계획

| 단계 | 기간(예시) | 목표 | 주요 작업 |
|------|-------------|------|-------------|
| 1단계: Prototype | Week 1~2 | 플레이 로직 완성 | 육각형 좌표계, 지뢰 배치, 클릭 판정, 플러드필 |
| 2단계: Alpha | Week 3~4 | 스테이지 로드 및 클리어 구조 | 맵 데이터 로드, 승패 판정, 리스타트 기능 |

| 4단계: Polish | Week 6 | 시각적 통일성 확보 | 파스텔톤 테마, HUD 다듬기, 사운드/이펙트 조정 |


## 7. 기술적 포인트

- 육각형 좌표계: axial (q, r) 시스템 사용  
- 플러드필: BFS 기반 0칸 자동 오픈 구현  
- 이징 시스템: ease_out_cubic 기반 스케일/알파 트윈  
- Add 블렌딩 효과: 파스텔톤 글로우 연출  
- 스테이지 데이터 구조: JSON 기반 (맵 크기, 난이도, 규칙 정의)

## 10. 예상 결과물 요약

지뢰찾기의 논리 쾌감과 은하 복원의 몰입감을 결합한 퍼즐 게임.  
플레이어는 중심 코어에서 시작해 육각형으로 퍼져 나가는 37개의 퍼즐을 정화하며,  
우주 속 데이터 질서를 되찾게 된다.
